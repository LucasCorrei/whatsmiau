Configuração para o SGP:

Texto simples

Qualquer texto sem tags é enviado como mensagem de texto.

[dividir_mensagem]

Separa o conteúdo em mensagens distintas (pausa entre envios).

[sendimg=URL][legenda opcional]

Envia uma imagem a partir da URL. Legenda é opcional.

Ex: [sendimg=https://site.com/foto.jpg]Veja a foto

[boleto=código,valor] ou [boleto=código,valor,tipo]

Exibe a linha digitável do boleto formatada.

Ex: [boleto=1234.5678 9012.3456,R$150,00,Boleto]

[link=URL] ou [link=URL,nome]

Se a URL servir um PDF, envia como documento. Caso contrário envia como texto (WhatsApp gera preview).

[linkcurto=URL]

Encurta a URL via serviço externo e envia o link curto.

[qrcode=conteudo]

Gera e envia um QR Code com o conteúdo informado.

Ex: [qrcode=https://meusite.com/pagar]

[lista]

Envia menu de preferência de recebimento de boleto (App / WhatsApp / Físico).

[template:nome;param1;param2]

Envia um template com nome e parâmetros separados por ponto-e-vírgula.

🆕 Botões de resposta rápida

[botoes=corpo|Btn1;Btn2;Btn3]
[botoes=título|corpo|Btn1;Btn2;Btn3]

Envia mensagem com até 3 botões de resposta rápida. Separe os botões com ;.

Ex: [botoes=Escolha uma opção|Sim;Não;Talvez]

[fim]

Agenda o envio para daqui a 5 segundos. Se for fim de semana ou feriado, agenda para o próximo dia útil.
